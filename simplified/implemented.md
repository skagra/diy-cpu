# Implemented OpCodes

## Done

* `ADC`
    add with carry
* `BEQ`
    branch on equal (zero set)
* `BNE`
    branch on not equal (zero clear)
* `DEX`
    decrement X
* `INX`
    increment X
* `JMP`
    jump
* `LDA`
    load accumulator
* `LDX`
    load X
* `SBC`
    subtract with carry
* `TAX`
    transfer accumulator to X
* `TXA`
    transfer X to accumulator
